Get started

    API Endpoint

    http://keypressingmonkey.pythonanywhere.com/istempemail/

This Temporary Email Checker API tells you whether the domain ending is part of a list of known spam / temporary email providers. This can help you prevent fake signups to various services.

To use this API, you need an API key from RapidApi, you can get one through this link:

get characters


http://keypressingmonkey.pythonanywhere.com/
istempemail/?email=YOUREMAIL

To get a return you need to make a GET call to the following url :
http://keypressingmonkey.pythonanywhere.com/ istempemail/?email=YOUREMAIL. Take note: While the code automatically strips off anything before the @-sign it is still recommended that you strip off the actual email address that comes before that. This is in your own interest, but not required. 



Result example :
#http://keypressingmonkey.pythonanywhere.com/
istempemail/?email=dfssd@byom.de
true

#http://keypressingmonkey.pythonanywhere.com/
istempemail/?email=dfssd@web.de
false

QUERY PARAMETERS

Field Type Description
email String E-Mail address you want to check.

Errors

The SPI uses the following, self-explanatory error codes:

Error Code solution
No email address passed with call Your GET-call is wrong somehow. Correct would be: http://keypressingmonkey.pythonanywhere.com/istempemail/?email=YOUREMAIL
invalid authentification IP address Only IP addresses whitelisted through RapidAPI are allowed. Sign up to the API at rapidApi.com and they will supply you with everything you need to sign up.
only GET allowed Since this API does not need any POST, UPDATE or DELETE requests I have disabled those. If you get this error your call is wrong, make sure you use get.
